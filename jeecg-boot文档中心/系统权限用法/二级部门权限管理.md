# 二级部门权限管理
> 二级部门权限管理主要是为部门管理人员提供设置部门角色、分配部门权限的功能。

[TOC]
## 1. 首先给部门授权
![](images/7e6b692b922244acf7c724c476734609_1323x683.png)
在部门管理中，选中一个部门给部门授权
## 2. 添加部门角色
![](images/0388f0605e15885b05113b041ce5a821_1377x592.png)
在我的部门-部门角色中添加部门角色
## 3. 给部门角色授权
![](images/23591e699ceaed35259992bc1305c20e_886x481.png)
在我的部门-部门角色中给所添加的部门角色授权
## 4. 用户管理中将用户设置部门、身份设为上级、设置负责部门
![](images/29f77371c4c05b0ac5a7595778b1469b_670x471.png)
## 5. 在我的部门-用户信息中为用户分配部门角色
![](images/c8183e24b14381e14860543c3c67fdbe_1313x602.png)
分配好部门角色以后，用户再登录时就有所设置的权限了
