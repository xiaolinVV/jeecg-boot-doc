**样式问题：** 
`使用代码生成器生成主子表，一对一的子表有字段是日期或者下拉框的情况，表单页面可能会出现被遮挡的效果`
**如下：**
![](images/e8680f1d6b714dbc31439c4e95e61ed0_1341x282.png)

**优化步骤：**
1.给弹窗modal添加一个class
![](images/bc6970718cc7f7d38bc87d7c57b2c9ce_674x213.png)
~~~
:class="'jee-outer-modal'"
~~~

2.在子表的form页面添加method：
![](images/5d8ca9e9f84fd6cab4d3ee0de58c773f_856x116.png)
~~~
getParentModal(){
  return document.querySelector('.jee-outer-modal')
},
~~~

3.在日期框或者下拉框上添加属性：
![](images/754db10d4befee21ad535f7eee562617_1250x237.png)


~~~
日期框
:getCalendarContainer="getParentModal"
~~~

~~~
下拉框
:getPopupContainer="getParentModal"
~~~
