# 初始化表单默认值
>[info] 功能说明：在表单开始加载，弹出表单时，即设置表单字段默认值。

1.使用`“loaded”`方法
|  方法   |   描述  |
| --- | --- |
|  loaded  |   页面加载时执行  |

2.选中所需单表或主表，点击`“js增强”`，定义表单js增强
![](images/691e354f761f53e202d18d5deeab42e1_1000x345.png)
代码如下：
```
loaded  (){
  that.$nextTick(() => {
   //age是对应表的字段名
    that.form.setFieldsValue({"age":"age值"})
  });
}
```

3.功能测试，点击添加，发现年龄设置默认值成功
![](images/65ff17fce5ebc84dc709552d17de1701_1633x759.png)
