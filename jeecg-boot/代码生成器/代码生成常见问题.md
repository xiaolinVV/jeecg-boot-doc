代码生成器—常见问题
===

## 1、生成代码为空
解决： 确认下代码路径是否有空格或者中文，去掉空格或者中文就好了
    比如：E:/workspace/jeecg-boot-framework
![](images/0d85a2470e46534920d73d5b07c8fe87_731x455.png)

##  2、JAR方式发布项目后，代码生成器不能用
http://doc.jeecg.com/2043922

##   3、微服务部署生成代码有问题
代码生成器依赖升级到1.3.1+
```
<dependency>
  <groupId>org.jeecgframework.boot</groupId>
  <artifactId>codegenerate</artifactId>
  <version>1.3.7</version>
</dependency>
```

## 4、如何支持 postgresql模式
代码生成器依赖升级到1.3.4+
```
<dependency>
  <groupId>org.jeecgframework.boot</groupId>
  <artifactId>codegenerate</artifactId>
  <version>1.3.4</version>
</dependency>
```

配置文件jeecg/jeecg_database.properties增加属性
```
#PostgreSQL模式，多个逗号分隔
schemaName=public,test
```

## 5、freeMark错误
 如果生成代码里面有freeMark错误，则表示生成代码有问题（虽然提示成功，但是代码生成是有问题的），需要排查Online表单配置是否有问题，一般是缺少字典配置
