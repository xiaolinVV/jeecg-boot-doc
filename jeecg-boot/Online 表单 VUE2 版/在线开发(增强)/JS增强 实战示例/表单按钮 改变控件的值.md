===
>[info]  本文主要讲述通过新增自定义表单类型的按钮 改变表单的值

## 配置步骤
- 新增自定义按钮，按钮编码为`“cc”`、按钮样式为`“form”`、按钮类型为`“js”`，如下图：
![](images/769b709b9fa8e408ea24388d15fac8b5_955x716.png)
- 新增表单JS增强方法，方法名为`“cc”`，与自定义按钮编码`“cc”`一致(**必须**)
![](images/50feaf578d4dbf24fe24e07fb25d949c_1012x357.png)
## js代码示例
```
cc(){
	that.triggleChangeValue("jiguan","汉族");
}
```

## 注意事项
1.that 指向当前vue实例，相当于vue中的this，可以调用一切this可以调用的方法，属性
2.此处调用的是triggleChangeValue，方法有两个参数，第一个对应字段名称，第二个是需要设置的值，目前只支持**单表/主表** 值的设置
