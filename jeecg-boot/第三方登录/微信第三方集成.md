查看application.yml  **justauth.type.WECHAT_OPEN** 配置，需要配置三个信息
![](images/ecfb635c11cf0d8b04a46abc8f56f66d_1253x150.png)

进入微信开放平台点击管理中心，进入网站应用
![](images/3438dd1f76ddee05c3f8a3ab769cbe8e_1094x200.png)

此过程需要网站应用，如果没有需要创建
![](images/629d564e742fb7d7af237a3d5e47a73f_1585x439.png)
创建成功后进入查看获取到appid和AppSecret，并将之填写到对应的yml中
 ```
client-id -> appid
client-secret -> AppSecret
```
![](images/2c526eb0d9b6257720cccb32920d4949_1349x225.png)

修改回调地址
网页滑到最下方，点击修改即可
注意：回调地址不用带http和https
![](images/16b9dbba3b25b7ca500344c712578ccf_1606x221.png)
将回调地址填入yml中即可
![](images/533188abfdf7749e62a24cfb01959f0c_1089x51.png)
